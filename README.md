# Heurística de busca em vizinhança variável para o problema de múltiplas mochilas com conflitos entre categorias e itens compulsórios 

* Diciplina PTR5001 da Escola Politécnica da USP - Oferecida pelo Claudio Barbieri 


** RESUMO

Problemas de Múltiplas Mochilas com Conflitos e Itens Obrigatórios é uma derivação do Problema de Múltiplas Mochilas,
 em que um conjunto de itens de diferentes pesos e valores devem ser alocados em um certo número de mochilas respeitando
 a suas capacidades. As diferenças do problema trabalhado é a existência de itens que não podem ser colocados na mesma mochila, 
pois eles são incompatíveis, e itens que são obrigatórios. É um problema comum na distribuição física de produtos. 
O objetivo deste trabalho é introduzir duas novas variantes para o Problema de Múltiplas Mochilas, e apresentar uma heurística baseada em 
Busca em Vizinhança Variável para resolver o problema.
